### Analysis of AI Agent Activity

I analyzed AI agent activity from the AIDev Dataset (MSR 2026 Mining Challenge) to answer key questions such as:
- How frequently do Coding Agents contribute tests? What types (e.g., unit, integration, end-to-end) are most common?
- What is the test-to-code churn ratio across ecosystems?
- When tests are missing in initial Agentic-PRs, do developers intervene to ensure reliable software testing (via follow-up commits or related PRs)?

Tools used: Python, pandas, visualization libraries 

🔗 [Repository](https://github.com/rmkamphuis/DATA-542-Project.git)

### R Wrapper of the Metropolitan Museum of Art Collection API - metExplore

I contributed to building an R wrapper that makes the Metropolitan Museum of Art Collection API easier to use. 
- Search by keyword and filters
- Retrieve metadata for one artwork or many artworks
- Returns consistent tibbles (minimal but useful wrangling)

🔗 [Repository](https://github.com/ChoWingDev/Data534_Museum_of_Art_API.git)

### Predicting Titanic Passenger Survival

I built a Decision Tree model to predict passenger survival on the Titanic, while my teammates developed Logistic Regression and K-Nearest Neighbors models.
- Focused on feature selection, model training, and performance evaluation
- Compared results across models to understand which factors most influenced survival
- Applied Python, pandas, and scikit-learn

🔗 [Repository](https://github.com/rmkamphuis/DATA_572_Project.git)
